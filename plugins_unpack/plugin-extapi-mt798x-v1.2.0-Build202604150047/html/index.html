🧩API 接口说明：
接口名称：addSK5Node
接口功能：添加SK5节点接口，若相同名称的节点已经存在则更新该节点。节点更新后直接生效，无需重启服务。
接口地址：POST  {{window.location.origin}}/api/call
请求参数：{
"auth_token": "{{ token }}",
"action": "addSK5Node",
"param": {
"name": "sk5node01",
"address": "110.110.110.110",
"port": "1081",
"type": "socks5",
"user": "username",
"password": "password",
"interfacename": "",
"dialer": ""
}
}
返回结果：{
"Result": 30000,
"ErrMsg": "Success"
}
补充说明：若需指定出口线路则请求参数传入interfacename，若需指定中转节点则请求参数传入dialer，否则传空字符串。interfacename和dialer不能同时传入。
接口名称：delSK5Node
接口功能：删除指定SK5节点接口，按节点名称匹配删除。
接口地址：POST  {{window.location.origin}}/api/call
请求参数：{
"auth_token": "{{ token }}",
"action": "delSK5Node",
"param": {
"name": "sk5node01"
}
}
返回结果：{
"Result": 30000,
"ErrMsg": "Success"
}
接口名称：addSK5Rule
接口功能：添加SK5分流规则，按终端IP地址匹配，若存在已有记录则更新规则。
接口地址：POST  {{window.location.origin}}/api/call
请求参数：{
"auth_token": "{{ token }}",
"action": "addSK5Rule",
"param": {
"address_ip": "192.168.9.100",
"name_sk": "sk5node01"
}
}
返回结果：{
"Result": 30000,
"ErrMsg": "Success"
}
