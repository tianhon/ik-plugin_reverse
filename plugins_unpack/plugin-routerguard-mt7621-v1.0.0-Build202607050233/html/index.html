使用提示：
1、开启后会自动收敛路由器本机入口，尽量只保留必要的基础网络能力。
2、可信 IP 和 MAC 默认尝试取当前访问来源；应用后必须在 180 秒内点击“确认保留”，否则自动回滚并关闭防护。
3、此功能用于快速收敛未知入口，不能清除已存在木马，建议配合强密码和 ARP/IP-MAC 绑定。
